# <a name="recommended-adapters"></a>Adaptadores recomendados
> [!NOTE]
> atualizaremos, fazendo adições e podem ser revisando essas Windows Mixed Reality recomendações de adaptador de PC. Consulte novamente regularmente para obter as últimas novidades.

seu [PC Windows Mixed Reality compatível](windows-mixed-reality-minimum-pc-hardware-compatibility-guidelines.md) pode não ter as portas HDMI e USB 3,0 de tamanho completo necessárias para conectar o headset de imersão. Se esse for o caso, você precisará de um adaptador. Alguns adaptadores também exigem a adição de um acoplador para se conectar ao seu PC ou ao cabo do headset. Listamos os diferentes tipos de adaptadores disponíveis e algumas recomendações sobre modelos específicos para você começar.

## <a name="windows-mixed-reality-compatible-usb-bluetooth-adapters"></a>Windows Mixed Reality adaptadores de Bluetooth USB compatíveis

Se o headset tiver sido lançado após 2018, você não precisará de um par de adaptadores e controladores com o headset.

Os dispositivos enviados em 2017 ou anteriores precisam de um adaptador para controladores. Se o computador não incluir Bluetooth para emparelhar controladores de movimento Windows Mixed Reality [,](controllers-in-wmr.md#pair-motion-controllers)você poderá usar um adaptador como o Adaptador de Microesporte usb Bluetooth 4.0 de baixa energia.

> [!NOTE]
> A Microsoft não fabrica, testa, certifica nem garante a funcionalidade, a confiabilidade ou a qualidade de qualquer adaptador. Para dúvidas, preocupações ou problemas com qualquer adaptador, entre em contato com o vendedor ou o fabricante do adaptador.

## <a name="using-external-usb-30-hubs-with-windows-mixed-reality-headsets"></a>Usando hubs USB 3.0 externos com Windows Mixed Reality headsets

Você pode usar Windows Mixed Reality headsets com um hub USB 3.0 externo se o computador não tiver portas USB abertas suficientes. O hub USB 3.0 externo deve ser alimentado externamente por meio de um adaptador **ac** para fornecer energia suficiente para o headset.

Observe também as seguintes diretrizes para uso de um hub USB 3.0 externo: 
* O hub USB deve ser um hub USB 3.0.
* O hub USB 3.0 deve estar conectado a uma porta USB 3.0 em seu computador.  
* O hub USB 3.0 deve ser alimentado externamente (ter um adaptador AC que fornece energia extra).
* O Hub SuperSpeed usb 3.0 de 10 portas plugável com adaptador de energia de 48 W é conhecido por funcionar bem com headsets Windows Mixed Reality e a maioria dos PCs.
